**this is for if you like all your groupchats and text your loved ones on instagram but want to stop doomscrolling**

_**üì¶ Installation**_

**Step 1: Create the Script**
Open the Scriptable app.
Create a new script and name it
Paste the code in script.js from this repo

**Step 2: Create the Shortcut**
Open the Shortcuts app on your iPhone.
Tap + to create a new Shortcut.
Add the action "Run Script" (search for Scriptable).
Select your InstaChat script.
Crucial: Tap the arrow next to the script name and toggle Show When Run to ON.

**Step 3: Add to Home Screen**
Inside the Shortcut editor, tap the Share icon (or the info i icon at the bottom).
Select Add to Home Screen.
Rename it to "Instagram" (or whatever you prefer).
Tap the icon to replace it with a custom Instagram icon (download a PNG from Google Images).
Tap Add.
